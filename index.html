Idea Network Analysis is a method for crowdsourcing the understanding of and the design of interventions for any number of complex challenges. Unlike other group processes which rely on consensus; this process elicits the many and diverse viewpoints on an issue. The methodology relies on brainstorming, sorting, and rating to collect input from the group. Multivariate analyses (multidimensional scaling, hierarchical cluster analysis) then create a visual representation, a map, of the group's thinking. Included here are several programs that support the analysis and display for Idea Network Analysis.

Reshape data for MDS

R tools that will reshape sorting data in into a square and symmetric summary matrix suitable for multidimensional scaling for Idea Network Analysis.

  • R tool to reshape racked data into matrix
  • R tool to reshape stacked data into matrix

R code for Idea Network Analysis

R-code that reads a summary matrix of sort data to create a 2D map of items. Outputs include an excel file of cluster membership for five to 15 clusters and a powerpoint file of maps that can be used to create presentations.

  • R code for concept mapping

Pattern Matching using Ladder Graphs

R-code that creates a ladder graph to show the pattern of values among clusters or ideas within a cluster. This type of graph is similar to the “parallel coordinates” graph, an important tool in the field of data visualization.

  • R code for pattern matching
