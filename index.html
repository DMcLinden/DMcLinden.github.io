Concept mapping is a method for crowdsourcing the understanding of and the design of interventions for any number of complex challenges. Unlike other group processes which rely on consensus; concept mapping elicits the many and diverse viewpoints on an issue. The methodology relies on brainstorming, sorting, and rating to collect input from the group. Multivariate analyses (multidimensional scaling, hierarchical cluster analysis) then create a visual representation, a map, of the group's thinking. Included here is the code to create a concept map using R as well as images of several maps.

Reshape manual data entry for MDS

An R tool that will reshape manual data entry of sorting data in into a square and symmetric summary matrix suitable for multidimensional scaling for concept mapping.
